Summary

📌 What is This?

Real-time checkbox synchronization app built with WebSocket technology. Changes sync instantly across all connected clients.

⚡ Key Features

  • Google OAuth2 Authentication
  • Real-time sync via Socket.IO
  • Redis state persistence
  • Rate limiting (1 sec cooldown)
  • Live timer display
  • Error notifications
  • Responsive design

🛠️ Tech Stack

  • Backend: Node.js, Express
  • Auth: Google OAuth 2.0
  • Real-time: Socket.IO
  • Cache: Redis / Valkey
  • Frontend: HTML5, CSS3, JS

🎯 How It Works

  1. Login with Google
  2. Load 100 checkboxes
  3. Click to toggle state
  4. State sent to server
  5. Syncs to all clients
  6. Saved in Redis

⚠️ Rate Limiting

Changes limited to 1 per second per user to prevent spam. Red error alert appears if limit exceeded.
