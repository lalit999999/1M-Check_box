Check Boxes

Summary

📌 What is This?

Real-time checkbox synchronization app built with WebSocket technology. Changes sync instantly across all connected clients.

⚡ Key Features

  • Real-time sync via Socket.IO
  • Redis state persistence
  • Rate limiting (3 sec cooldown)
  • Live timer display
  • Error notifications
  • Responsive design

🛠️ Tech Stack

  • Backend: Node.js, Express
  • Real-time: Socket.IO
  • Cache: Redis
  • Frontend: HTML5, CSS3, JS

🎯 How It Works

  1. Load 100 checkboxes
  2. Click to toggle state
  3. State sent to server
  4. Syncs to all clients
  5. Saved in Redis

⚠️ Rate Limiting

Changes limited to 1 per 3 seconds per client to prevent spam. Red error alert appears if limit exceeded.
